Recherchez une ville

Ce système d'auto-completion fonctionne avec des requête Ajax vers un fichier PHP qui trouve les correspondances dans un ensemble de villes.
Il est possible de se déplacer avec les flèches haut et bas et de sélectionner une ville dans le menu déroulant qui affiche les propositions.
Seul le système d'auto completion est traité ici : La requête pour afficher les informations concernant la ville en question n'est pas menée ici.
